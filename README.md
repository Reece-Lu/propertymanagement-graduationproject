# 后疫情时代智慧物业管理系统
基于Vue2.X的前端项目，基于普通物业管理系统，有基本的业主档案统计与报修维修功能。面向疫情背景加入新的需求，有疫情防控和居家隔离帮助和管理，以及宠物寄养和挪车代驾等新需求。
## Project setup
```
npm install
```
